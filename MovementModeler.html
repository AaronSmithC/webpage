MovementModeler

Turn any video into clean, accurate stick-figure motion.

What Is MovementModeler?

MovementModeler is an on-device motion-analysis tool that converts videos into clean stick-figure motion clips. Created for both everyday creators and computer-vision professionals, the app offers smooth tracking, adjustable filters, multi-person handling, and exportable BODY-25 JSON data.

Key Features

  • Convert any video into clean stick-figure motion
  • Adjust smoothing, fill missing limbs, and reduce jitter
  • Multi-person tracking with ID stability
  • Export BODY-25 JSON pose data for CV/ML workflows
  • Export MP4 stick-figure animations
  • All processing is done on-device — nothing is uploaded

Who Is It For?

MovementModeler is built for creators, dancers, athletes, educators, engineers, CV researchers, and anyone interested in exploring human motion. No accounts, no servers, and no technical setup required.
